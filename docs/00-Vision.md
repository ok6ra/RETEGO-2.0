# 00 - Vision

## Mission
RETEGO Framework is the website foundation for a Czech company that designs and delivers integrated access and entry solutions for buildings, industrial sites, logistics centres and residential complexes.

## Core promise
Řešení tam, kde běžná nestačí.

## Business focus
The website must explain how RETEGO combines:
- access control systems
- vehicle entry control and LPR
- barriers, gates and door automation
- turnstiles and intercoms
- CCTV and video monitoring

## Product positioning
The brand must be presented as a practical systems integrator, not as a generic security reseller. Every page should communicate real installation logic, service scope and operating outcomes.

## Audience
Primary audiences are facility managers, operations directors, security managers, project investors and technical decision-makers.

## Content rules
- Write in clear Czech first.
- Use concrete terms such as object, vjezd, přístup, kamera, turniket, závora and SPZ.
- Avoid vague claims and filler copy.
- Link technical services to real operational benefits.
