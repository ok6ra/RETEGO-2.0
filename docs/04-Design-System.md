# 04 - Design System

## Visual direction
The website should feel modern, technical and trustworthy. The visual system should support security, infrastructure and industrial environments without looking overly corporate.

## Core palette
- Primary: #0B5FFF
- Secondary: #111827
- Background: #FFFFFF
- Light background: #F8FAFC
- Accent: #2563EB

## Typography
- Headings: Inter
- Body: Inter
- Enable fluid typography

## Layout rules
- Use an 8px spacing system
- Content width: 760px
- Wide content width: 1280px
- Support responsive spacing and block gaps

## Component rules
- Use clear section spacing
- Keep buttons visually simple and action-oriented
- Use cards for service and problem summaries
- Keep forms and CTAs easy to scan

## Gutenberg compatibility
The design system must support:
- appearance tools
- custom spacing
- custom border radius
- gradients
- duotone
- block gap
- responsive layouts

## Accessibility rules
- Maintain strong contrast
- Use semantic headings and landmarks
- Keep navigation simple and keyboard accessible
- Ensure images have meaningful alt text
